# Мониторинг статусов потоков (workflow_monitor)

Скрипт для автоматической проверки статусов потоков данных на серверах оркестратора.

## 📋 Описание

`workflow_monitor.py` позволяет:
- Проверять статусы нескольких потоков на нескольких серверах
- Получать информацию о времени запуска, ID, описании потоков
- Сохранять результаты в JSON, CSV и сводной таблице
- Работать как в режиме Python, так и в собранном EXE

## 📁 Структура

```text
src/monitoring/
├── workflow_monitor.py # Основной скрипт
│
configs/monitoring/ # Конфигурационные файлы
├── servers.txt # Список серверов
├── workflows.txt # Список потоков
└── config.txt # Настройки мониторинга

reports/ # Отчеты (создаются автоматически)
└── reports_YYYYMMDD/
├── report_YYYYMMDD_HHMMSS.json
├── report_YYYYMMDD_HHMMSS.csv
└── pivot_YYYYMMDD_HHMMSS.csv
```

## ⚙️ Конфигурация

### 1. `servers.txt` - список серверов

Формат: `URL,логин,пароль`

```txt
# Сервер 1
http://127.0.0.1,user,pass

# Сервер 2
http://127.0.0.2,user,pass
```

### 2. workflows.txt - список потоков

Каждая строка - имя потока:
```txt
# Основные потоки
wf_test_1
wf_test_2
```

### 3. config.txt - настройки мониторинга

```txt
# Задержка между запросами (секунды)
delay_between_requests: 0.5

# Задержка между серверами (секунды)
delay_between_servers: 2.0

# Таймаут для запросов (секунды)
timeout: 30

# Сохранять CSV отчет
save_csv: true

# Максимальное количество ошибок для показа в сводке
max_errors_display: 5
```

## 🚀 Запуск

### Из Python

```bash
cd ..\YamlGenerators
python src\monitoring\workflow_monitor.py
```

## 📊 Форматы отчетов

### 1. JSON (report_*.json)
Полные данные для машинной обработки:

```json
{
  "http://127.0.0.1": {
    "wf_test": {
      "name": "wf_test",
      "status": "CLOSED",
      "start_time": "2026-01-27 16:43:41.277489",
      "id": 285271070,
      "description": "Поток для тестовых запусков",
      "enabled": false,
      "domain": "abc",
      "found": true
    }
  }
}
```

### 2. CSV (report_*.csv)

Таблица для Excel (разделитель ;):

```csv
Сервер	    Имя потока	 Статус	  Статус (описание)	  Время запуска	        ID	        Описание	        Включен	  Домен
http://...	wf_test	     CLOSED	  Завершен	          2026-01-27 16:43:41	285271070	Поток для тестов	Нет	      abc
```
